パワプロ型キャバ嬢育成シミュレーション。設計の正は `GAME_DESIGN.md`。

## 遊び方

`index.html` をダブルクリックするだけ（ブラウザで動く。サーバー不要）。

## カレンダーと100日構造（v0.6）

- **12/20（金）スタート→Day100＝3/29**。翌3/30が推しのツアー初日。実在曜日つき（2024-25暦と一致）
- 目標：**通帳¥1,000,000**（頭金10万＋90万稼ぐ）。到達で早期クリア＝**達成日数がスコア**
- **毎月1日に生活費¥150,000引き落とし**（1/1・2/1・3/1）。残高マイナスあり
- **店休日**：12/31大晦日・**1/1元日=実家で「打ち明けるか」選択イベント**・1/2初詣。12/24-25はドリンクバック×2の繁忙イベント
- **流し出勤**：夜を自動処理（日給¥10,000＋α・消耗軽・指名関係は進まない）。本気の夜と流す夜の配分が戦略
- **大学**：土日・冬休み（12/26〜1/6）休講。**1月末までに10回出席**で進級。留年＝3/1に学費-¥100万を背負って続行

## 夜の構造（v0.6）

- 舞台は**八王子**。1晩＝一見さん2卓＋**指名候補1卓（ローテーション・夜の最後に登場）**
- 各卓：**探り3ラリー → 正答率60%以上で場内指名 → さらに5ラリー**（おねだり解禁）。**一見さん・変な客も3ラリー×毎回4択**（一客最低3ラリーの原則）
- 指名客5人：院長（ドンペリ級）・ご隠居（スパーク）・課長（シャンパン）・**IT社長（ドンペリ級・忖度すると冷める）**・**支店長（シャンパン・定年3週間前）**
- **エピソード制**：来店のたびに違う話（現在8話ぶん収録）。話が尽きた客は登場しない→一見さんの夜になる

## 昼コマンドのメニュー（v0.5）

- **プチ整形**：6パーツ（目/鼻/肌/輪郭/歯/体型、容姿=平均）×クリニック3択（¥30,000/70%〜¥90,000/99%）。失敗=効果なし+体力-20
- **筋トレ**：ランニング（無料）/ジム（¥3,000）/パーソナル（¥10,000）。体型＋体力上限ガチャ（質が段階UP、上限130まで）
- **読書**：マンガ/自己啓発本/小説/新聞。**知性とトーク力はここで上げる**（接客ノートは廃止。トークは夜の実戦+2でも育つ）
- **趣味**：**メンタル回復の最大は「推しの配信」（無料+50）**。サウナ+40/映画+38/ヒトカラ+35/カフェ+30は回復が一段小さいかわりに「話題の種」が付く＝金を払って夜の武器を仕込む枠
- **休む**：体力爆増（+40〜55）・メンタル微増（+10〜15）。一日、部屋から出ない
- **話題の種**：映画（単館系）・マンガ・小説・新聞を摂取すると話題を1つ仕込める→**好みが合う客の卓でT1後に発火、好感度+20＋機嫌UP**（使うと消費）。好み＝院長:新聞／課長:マンガ／ご隠居・支店長:小説／IT社長:映画
- 棲み分け：**整形・筋トレ→容姿／読書→知性・トーク／大学→知性**

## 3つのバロメータ（v0.4）

- **体力＝働ける量**：卓に着くたび-8、朝+15回復。20未満=出勤不可／夜中に尽きたら即退勤／40未満で出勤=発熱リスク30%（2日寝込む・要警告表示）。筋トレは体力を削って上限に変える（ランニング-8/ジム-18/パーソナル-30。高いほど重いが効率がいい）
- **メンタル＝接客の質**：凡打で流すのは安く、刺しにいくと削れる。20以下=選択肢に「疲れた心の一言」が混入／**10以下=最悪の選択肢しか浮かばない**
- **店長の信頼＝卓の質**：場内・ボトル・皆勤で上昇、欠勤・病欠・早退・爆発(-20)で下落。40未満=院長卓が変な客に／25未満=課長卓も／**5以下=解雇エンド**

## 給与体系（八王子の実相場ベース）

- **日給保証 ¥15,000**（新人時給3,000円×5時間相当）
- **ヘアセット代 -¥2,000**：出勤するたび給料から天引き（本気でも流しでも。夜職の固定費）
- **ドリンクバック ¥400/杯**：卓が長く続くほど杯数が増える（2ラリー=1杯）。イブ・クリスマスは×2
- **場内指名バック ¥1,000/回**
- **ボトルバック 22%**：スパークリング¥20,000／シャンパン¥80,000／ドンペリ級¥200,000 の店頭価格×22%
- **流し出勤は日給¥10,000＋ドリンク1〜3杯**（早上がり扱い）
- 「もっと稼げばもっと会える」＝都心ステップアップ編（DLC）の動機に接続
