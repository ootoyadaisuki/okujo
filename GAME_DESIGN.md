## 2. スコープ（v1 = MVP）

### v1に入れる
- 1シーズン（十数営業日）で完結。勝ち／負けの1エンドのみ
- ステータス3種: **容姿・知性・トーク力**
- 育成コマンド（v0.5でメニュー選択式に拡張）:
  - **大学**: 知性+8・体力-12。出席は進級の義務（§1「大学と留年」参照）。1月末（Day10）を過ぎると選べない
  - **プチ整形**: 容姿は6パーツ（目・鼻・肌・輪郭・歯・体型、各20開始）の平均。パーツを選ぶ→クリニック3択（BeBe美容外科 八王子駅前院 ¥30,000/+18/成功70%、ルミナス美容クリニック ¥50,000/+24/90%、聖蘭記念美容外科 ¥90,000/+30/99%）。失敗＝効果なし＋体力-20（ダウンタイム）。成功率は必ず%表示（破滅は読める原則）。※高級院を院長or後輩の院にする伏線イベントはv1.1に温存。メニュー層（埋没vs切開等）はv2
  - **筋トレ**: ランニング（無料/体型+4/上限+1〜2）、ジム（¥3,000/+8/上限ガチャ+1=50%,+2=35%,+3=15%）、パーソナルトレ（¥10,000/+12/上限+2〜4）。体力上限は130まで
  - **読書**: 知性とトーク力の供給源。マンガ（¥500/知+1/トーク+4/心+10）、自己啓発本（¥1,500/知+3/トーク+6/心+4）、小説（¥800/知+6/トーク+3/心+6/体-3）、新聞（¥200/知+9/トーク+2/心-3/体-5）
  - **趣味**: 推しの配信（無料/心+20/今夜バフ）、推しの現場（¥15,000/心+70/体力-10/今夜バフ）、ヒトカラ（¥1,500/心+25/体力-5）、映画（¥1,900/心+30/知性+2）、カフェ（¥1,200/心+15/体力+10）。**今夜バフ（客の好感度×1.2）は推し系のみ**
  - **休む**: 体力・メンタルとも+30〜50のランダム回復。25%で彼氏イベント（+10）
  - **接客ノートは廃止**。ステータスの棲み分け＝**プチ整形・筋トレ→容姿／読書→知性とトーク力／大学→知性／夜の実戦→トーク力+2**（※「接客ノート」の名前はv1.1の指名客・会話記憶システムの対策UIとして復活させる候補）

- 指名客5人（院長・ご隠居・課長・IT社長・支店長）。**エピソード制＝来店のたびに違う話**（院長ep2=ポルシェと息子の受験／課長ep2=約束の大阪編／ご隠居ep2=ひ孫誕生）。エピソードが尽きた客は新しい話ができるまで来ない
- 1晩＝一見さんの軽い卓2つ＋**指名候補1卓（ローテーションで選出・夜のクライマックスに配置）**

## 3. ゲーム全体のループ

1. 昼パート: 育成コマンドを選択（ステ上げ／メンタル回復／推し活＝回復＋バフ）
2. 夜パート: 出勤。指名客の卓を回る（1晩2〜3枠の「意味のある卓」）
3. 接客でシャンパン等を売り、バックを推し活資金として積み立てる
4. 接客はメンタルを削る。メンタルが尽きると出勤不可・パフォーマンス低下
5. 期限までに目標金額に到達すればクリア

### 席回りの設計思想
- 新人期＝高回転・浅い接客（誰が食いつくか探す）、成長後＝少数・深い接客（太客を育てる）
- ヘルプ・新規回しの大量の卓は軽い1〜2択の短い遭遇で流し、フルのターン制接客は「意味のある卓」だけに発生させる（実装負荷とプレイヤーの疲労を抑える）
- リアル考証: 新人はヘルプ＋フリー客からスタート、1卓に3〜4人がチェンジで入る回転。場内指名が付くと卓が深くなる

### 客のトリアージ（上層メカニクス）
- 席（時間）は有限。どの客に張るかの配分ゲーム
- 客ごとに成長曲線がある。最初渋い客が後に太客化することもあれば、そこそこ使う客に執着すると最終目標に届かない
- 「執着＝機会損失」を実在させる。外れても「あの判断が敗因」と後から読める学習可能な失敗にする

## 4. 接客本番のシステム（コア）

### 基本構造: 1接客 = 5ターン
- **T1〈入り〉**: 客の口上→最初の4択。比較的安全な話題
- **T2〜T3〈本題〉**: 自慢・話題が深化。空ヨイショの効きが落ち、的確な承認が正解になる
- **T4〈地雷原〉**: 危険話題に接近。最大の地雷を配置
- **T5〈クロージング〉**: おねだり判断＋締め

### メーター
- **好感度** 0〜100（開始30）: 客ごと。シャンパンおねだりの条件
- **体力＝働ける量**: 卓に着くたび減る（会話の質と無関係の肉体コスト）。20未満=出勤不可（強制欠勤）／夜中に尽きたら即退勤／40未満で出勤すると発熱リスク30%（→2日寝込む）。リスクは必ず事前警告（破滅は読める原則）。朝の自然回復あり。**筋トレで上限が伸びる（+1=50%/+2=35%/+3=15%、容姿+6と併給、上限130まで）**
- **メンタル＝接客の質**: 客に削られる。**40未満で選択肢に「疲れた心の一言」（ハズレ/地雷）が混入、10以下で正解・凡打が消えて最悪の選択肢しか浮かばない**（数値ペナルティでなく選択肢そのものが壊れる＝見える形の原則）。回復は休む/推しの配信/現場
- **店長の信頼＝卓の質**: 場内指名・ボトル・皆勤で上昇／欠勤-8・病欠-10・早退-6・客爆発-20で下落。40未満=院長卓が変な客に差し替え／25未満=課長卓も／5以下=**解雇（ゲームオーバー）**。「働く（体力心を削って信頼と金を得る）vs 休む（回復するが信頼が痩せる）」の三つ巴を作る
- **機嫌（裏パラメータ）**: 好/普/険 の3段階。プレイヤーには直接見せず、セリフ・表情差分で表現。知性が閾値以上ならテキストで開示（「…今日は機嫌が悪そうだ」）

### 4択の型（毎ターン共通）
1. **正解**: 好感度+15〜20／メンタル-5。客が本当に求めるものを刺す
2. **凡打**: 好感度+5／メンタル-5。空ヨイショ。T1では正解並みに効くが中盤から失速（学習曲線）
3. **ハズレ**: 好感度-10／メンタル-10。自分語り・話題逸らし・退屈な相槌
4. **地雷**: 下記参照。地雷ほど善意に見える文面で書く

### ステータスの効き方（「見える」形で）
- **知性**: 客の情報開示。客のタイプ/求めるもののヒント、地雷選択肢への⚠マーク（or 内心の警告テキスト）、機嫌の可視化。＝選球眼・地雷探知機・客を見る目
- **トーク力**: 正解・凡打の好感度上昇量アップ（パワー）
- **容姿**: 失敗時のペナルティ軽減（客が許す）＋色仕掛け系選択肢の効き。客によって容姿補正の効きが違う

### 地雷と爆発
- 地雷を踏む → 即出禁ではなく〈機嫌〉一段悪化＋好感度-25。声色が変わるセリフを挟む
- 機嫌〈険〉でさらに地雷 or ハズレ → **爆発イベント**（店で騒ぐ→黒服介入→出禁→この客の売上は永久ゼロ）
- 機嫌〈険〉直後のターンに特別選択肢**〈必死になだめる〉**出現: 好感度-15・メンタル-20の高コストで機嫌を〈普〉に戻す回避の窓
- 原則: 破滅は「読める・避けられる」ものにする。予兆の階段（機嫌悪化の描写、先輩の耳打ち、知性による警告）を必ず置く。理不尽な運ゲーにしない

### おねだり（シャンパン）
- T3以降、毎ターン終わりに「おねだりする／まだ待つ」の判断
- 好感度60=スパークリング（小）、80=シャンパン（中）、95＋機嫌〈好〉=ドンペリ級
- 足りない状態でねだる→失敗、好感度-15＋機嫌一段悪化（爆発の導火線にもなる）
- T5で終了＝待ちすぎたら今夜はゼロ。push-your-luck構造

### リプレイ性
- 初見はほぼ確実に大地雷を踏む設計でよい（「この世界には地雷がある」という学習）
- 2周目以降、接客前に先輩の耳打ちイベント（知性依存）で予習可能。同じ客が知識の有無で別難度になる

## 5. 1人目の痛客（チュートリアルボス）

### プロフィール
- 50代男性、美容クリニック経営者（医師）、年収1億円
- 医師家系出身。医学部卒→医師免許取得→**稼ぎを求めて**大手化粧品メーカーに研究者として就職（新人医師より稼げるため）→さらに稼ぎを求めて美容クリニックを開業
  ※実在の知人がモデルだが丸ごとにはしない。固有名詞（企業名等）はゲーム内に出さない
- 車: フェラーリ1台、ランボルギーニ1台、ゲレンデヴァーゲン1台
- 妻: 若い元キャバ嬢（キャバクラで出会った）。小学校低学年の子が1人
- クリニックは八王子で1・2位を争う有名院。だが**断トツではない**ことがコンプレックス
- ライバルの上位院を経営するのは**彼の後輩**。会えば先輩風を吹かせるが内心「負けたくない」。後輩も表面上は立てつつ闘争心を燃やしている
- 後輩の院は**駅のホームに巨大広告**を出している。彼の院は出していない。毎日視界に入る「#2の証」
- 馴染みのフレンチにクリニックのスタッフやパパ友を連れて行き、常連ムーブ（横柄だが許される）をするのが至福
- 周囲（店員・スタッフ・夜の店）からは**腫れ物扱い**されているが、本人はそれを敬意と誤読している。実はかなり寂しい
- 夜の店では有名人だが、業界では全員が要注意客と認識。**新人の主人公はそれを知らない**

### 心理の核
- 自己像: 「俺は誰より合理的に、正解を選び続けた勝者」。金は賢さの証明
- ガチの臨床医道を進んだ医師には**優越感**を持っている（引け目ではない）
- 深い傷: 「全部正しく選んで勝ったのに、誰も本気で自分を好きじゃない」
- 最高好感度で仮面が落ちる瞬間の本音: 「あいつ（後輩）にだけは、負けたくないんだ」「駅の広告、あれ、俺のじゃないんだ」

### ゲーム上の役割
- 金払いが非常に良く、売上の柱として頼りたくなる**高配当・高テールリスク客**
- 些細な地雷で店で騒ぎ、出禁→売上ゼロになりうる（集中リスク・一本足打法の危険を教える）
- 依存するほど地雷が増える設計も可: 主人公が他の卓に時間を割くと嫉妬して機嫌が悪化
- 正解を選んでもメンタルがよく削れる（見下しに耐える接客）
- 出禁メカニクス・予兆・回避のチュートリアルボス

### 褒めのルール
- **安全に褒めてよい**: 車、年収、経営者としての成功（表のフレックス）
- **正解の質**: 空ヨイショではなく「戦略的に賢かった」の具体的承認（メーカー行きも美容転身も先を読んだ判断だった、と彼の戦略眼を理解する。ただし本人が会話で出していない経歴を選択肢が先回りして知っていてはいけない）。彼が欲しいのは感嘆より「賢さの理解」
- **地雷**:
  - ガチ臨床医を「立派」と称賛する（彼の選択が劣ると暗に言うことになる）
  - 彼の成功を「商売上手」と矮小化する
  - 八王子のクリニック序列・ライバル院・後輩の話題
  - **最大の地雷**: 「駅のホームで大きな広告見ました！」（善意の褒めだが、あれは後輩の院の広告）
  - 腫れ物扱い（＝金で丁寧に扱われているだけ）を悟らせること

### T1の叩き台（要・本人の筆でリライト）
口上（案A・採用方向）:
「いやー、参ったよ。今日ゲレンデで来たんだけどさ、この辺コインパーキング狭すぎない？フェラーリじゃなくて正解だったわ。……ん、君、新しい子？」
- ポイント: 聞かれてないのに自慢が始まる／文句の形を借りた自慢（謙遜フレックス）／車が先、人間は後

4択の叩き台:
1. 正解: 「えっ、ゲレンデ"で"来た、ってことは……他にも？」（褒めるのではなく**続きを引き出す**。彼が欲しいのは発表時間の延長）
2. 凡打: 「かっこいいですね〜！」（T1では+10程度効くが後半失速する飛び道具）
3. ハズレ: 「私、車ってよくわからなくて…」（会話の受け取り拒否）
4. 悪手: 「ゲレンデって維持費すごいんですよね？知り合いも乗ってて」（他人を彼のステージに上げた。T1に本物の地雷は置かず、ここで「自分語り・他人語りがNG」を軽く学ばせる）

## 6. 設計原則（実装時に守ること）

1. **ステータスは見えない確率ではなく、プレイヤーの操作・情報を見える形で強化する**（パワプロの原則）
2. **破滅は読める・避けられる**。予兆→回避の窓→爆発、の階段を必ず踏む
3. **選択肢はどれも一見悪くなさそうに書く**。地雷ほど善意に見せる
4. **正解＝客が本当に求めているものを刺すこと**であり、単に褒めることではない
5. **稼ぎとメンタルのトレードオフ**が本番と育成をつなぐ背骨
6. **残酷なくらい小さく作る**。v1の完成・リリースが最優先。拡張はパーキングリストから
7. セリフ・テキストは作者本人（プロの物語書き）が書く。実装側は構造・数値・分岐を組む
